Биквадрат теңдеулер

8-сынып Алгебра: Мастер-класс

0 Бұл не?

Кәдімгі квадрат теңдеуді білесің ғой? Мысалы: ax² + bx + c = 0.

Ал Биквадрат (bi - екі деген мағынада) — бұл "екі еселенген" квадрат теңдеу.
Оның түрі:

ax4 + bx2 + c = 0

1 Басты құпия: "Жаңа айнымалы"

Біз x⁴-ті шеше алмаймыз, бірақ біз оны алдай аламыз. Біз x²-ті басқа әріппен (мысалы, t) алмастырамыз.

Ескі дос:

x⁴ + 5x² - 6 = 0

Жаңа кейіпкер:

t² + 5t - 6 = 0

МАҢЫЗДЫ: t әрқашан 0-ден үлкен немесе тең болуы керек (t ≥ 0), өйткені квадрат сан теріс болуы мүмкін емес!

Мини-ойын: Масканы тап!

Егер x⁴ - 13x² + 36 = 0 берілсе, t арқылы алмастырғанда теңдеу қалай өзгереді?

Жеңіс алгоритмі (Cheat Sheet)

1.

x² = t деп белгілеп ал. (t ≥ 0)

2.

Квадрат теңдеуді шеш (Дискриминант немесе Виет).

3.

t-ның мәндерін тап. Теріс мәндерді лақтырып таста!

4.

Кері қайт: x² = t деп шешіп, x-ті тап.
